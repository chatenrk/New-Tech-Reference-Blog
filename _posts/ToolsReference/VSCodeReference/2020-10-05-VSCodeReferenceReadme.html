# Extension Description
1 Markdown Preview Enhanced This extension provides you with many useful functionalities such as automatic scroll sync, math typesetting, mermaid, PlantUML, pandoc, PDF export, code chunk, presentation writer, etc.

Mainly useful for markdown files reading and writing
2 Bracket Pair Colorizer 2 Allows matching brackets to be identified with colors. The user can define which tokens to match, and which colors to use
3 .gitignore Generator Lets you easily and quickly generate .gitignore file for your project using gitignore.io
4 YAML Provides comprehensive YAML support to VS Code

7 Prettier This extension provides formatting for code written in vscode
