Using postman to query SAP Gateway

Determine the HTTP Address of the server.

  • Open T-Code – SMICM – > in the menu Go To-> Service
  • Note the HTTP Port Name and Host name. Ping the host name in Command Prompt to get the IP address
  • Use the IP Address and port from above to query the gateway system
